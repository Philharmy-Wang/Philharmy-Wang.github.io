# Guanbo Wang Academic Homepage

This repository hosts the GitHub Pages personal academic homepage for **Guanbo Wang / 王冠博**.

The site is a lightweight Jekyll-based academic homepage with:

- English default pages at `/`
- Chinese pages under `/zh/`
- Research, publications, projects, datasets and code, awards, and CV pages
- No paid services, external analytics, Disqus, or private/internal links

## Local Preview

```powershell
bundle install
bundle exec jekyll serve
```

If Ruby and Bundler are not installed locally, GitHub Pages can still build the site from the Markdown, layouts, includes, and static assets in this repository.

## Theme Acknowledgement

This site keeps a lightweight, vendored Jekyll theme structure adapted from Minimal Mistakes. It is published with GitHub Pages and powered by Jekyll.
